About

I myself am still learning but I made this website so I could summarize the information I have already learned quickly and efficiently so this is not going to be similar to a book but more or less just some facts, examples, etc. I don't plan on making this site pretty or very dynamic because it's not my end goal. I'm interested in hardware and software interacting with hardware, web development is not my area.
